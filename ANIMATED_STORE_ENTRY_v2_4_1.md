# ALIN v2.4.1 — Animated Store Entry

تمت إضافة شاشة بداية متحركة قبل دخول المتجر من دون تغيير تصميم المتجر أو وظائفه.

## التنفيذ
- شاشة افتتاحية متجاوبة للكمبيوتر والموبايل مستوحاة من التصميم المعتمد.
- حركة خفيفة للشعار واللمعة وشريط التقدم باستخدام CSS فقط.
- مدة ظهور محدودة: نحو 1.55 ثانية، ثم انتقال ناعم للمتجر.
- تجهيز رابط واجهة المتجر في الخلفية عبر prefetch أثناء عرض الشاشة.
- تحويل صور الشاشة إلى WebP خفيف: أقل من 29KB لكل نسخة.
- دعم تقليل الحركة من إعدادات الجهاز.
- تشغيل شاشة البداية أيضًا عند فتح نسخة PWA المثبتة.
- لم يتم تعديل واجهة المتجر أو نظام الطلبات أو قاعدة البيانات.

## الملفات الجديدة
- `styles/alin-splash.css`
- `core/splash.js`
- `assets/images/alin-splash-desktop.webp`
- `assets/images/alin-splash-mobile.webp`
