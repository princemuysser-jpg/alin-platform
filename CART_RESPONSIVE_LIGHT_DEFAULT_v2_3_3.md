# ALIN v2.3.3 — Responsive cart and light default

- The checkout modal now sits above the mobile navigation.
- The checkout panel uses the device viewport instead of extending behind the bottom bar.
- Checkout details scroll inside their own area while the confirmation button remains visible.
- Desktop checkout uses a fixed confirmation area and independent details scrolling.
- The mobile-only “مواد تناسب مرحلتك” block was removed; desktop keeps it.
- Light mode is the default for new and existing system-mode sessions.
- Dark mode remains an explicit user choice in Options.
