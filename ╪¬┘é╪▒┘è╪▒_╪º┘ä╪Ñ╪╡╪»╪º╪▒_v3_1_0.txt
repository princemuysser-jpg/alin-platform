منصة آلين v3.1.0 — تقرير الإصدار الموحد

الحالة: إصدار موحد للنشر التجريبي المنضبط.
عدد ملفات JavaScript المفحوصة: 80
ملفات SQL داخل الحزمة: 1
مراجع الملفات المحلية المفحوصة: 25

التغييرات الأساسية:
- تثبيت بنية الطلبات والحسابات الموجودة في ملف SQL واحد.
- إعادة بناء صلاحية ملفات الملازم الخاصة بالاعتماد المباشر على مستخدم Supabase والحساب المرتبط به.
- المكتبة تفتح PDF فقط عندما يكون الطلب تابعاً لها ولم يصل إلى حالة مكتمل أو ملغي.
- المدرس يفتح ملف ملزمته فقط، والمدير يفتح جميع الملفات.
- تحديث جلسة الدخول تلقائياً قبل إعادة محاولة قراءة الملف.
- قراءة PDF من Blob مباشرة دون إعادة فتح رابط مؤقت داخل المتصفح.
- نسخة واحدة وملف SQL واحد، ولا توجد ملفات ترقيع منفصلة داخل الحزمة.

حدود الفحص:
- لم يتم تشغيل SQL على مشروع Supabase الحقيقي.
- يجب تنفيذ طلب تجريبي واحد بعد النشر للتأكد من توافق البيانات الحية.
