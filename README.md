# منصة آلين — Ultimate v1.0 Modular

نسخة مقسمة حسب الوظيفة، بدون أسماء إصدارات داخل ملفات التشغيل.

## الهيكل
- `core/`: النواة، الاتصال، الحماية، نظام التصميم.
- `admin/`: أقسام لوحة المدير؛ كل قسم له JavaScript وCSS مستقلان.
- `teacher/`: لوحة المدرس.
- `library/`: لوحة المكتبة والطباعة والحسابات.
- `courier/`: لوحة المندوب.
- `store/`: المتجر والسلة والتوصيل وواجهة الدخول.
- `assets/`: الصور والأيقونات فقط.
- `database/`: ملفات قاعدة البيانات النهائية.

تم الحفاظ على ترتيب تحميل الأكواد الأصلي حتى تبقى التعديلات المتتابعة متوافقة، مع تجميع كل قسم في ملف ثابت واحد بدلاً من ملفات تحمل أرقام الإصدارات.
