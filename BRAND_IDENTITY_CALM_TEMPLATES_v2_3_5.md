# ALIN v2.3.5 — Visual Identity Fix and Calm Templates

- Separated visual identity presets from the light/dark appearance setting.
- The branding module no longer writes to `data-alin-theme`; it uses `data-alin-brand-theme`.
- Brand colors now apply to store, admin, teacher, library, courier, and login surfaces through shared variables.
- Added seven calm presets: Alin Original, Sky, Sage, Sand, Lavender, Rose, and Graphite.
- Added responsive preview, asset removal controls, upload validation, and double-save protection.
- Added a dedicated responsive stylesheet for the identity editor.
- Added Arabic, Kurdish, and English interface translations for the new identity controls.
- No database migration is required.
