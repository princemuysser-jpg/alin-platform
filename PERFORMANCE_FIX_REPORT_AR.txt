منصة آلين v1.0.8 — إصلاح الأداء الآمن

تم:
1) توحيد رقم الإصدار داخل VERSION و core/config.js والواجهات.
2) تفعيل Service Worker حقيقي لتخزين الملفات الثابتة بعد أول زيارة.
3) منع تخزين بيانات Supabase الحية حتى لا تظهر بيانات قديمة.
4) تخزين مكتبات jsDelivr خارجياً بعد أول تحميل لتقليل أثر ضعف الإنترنت.
5) إضافة preconnect و dns-prefetch لتسريع بدء تحميل المكتبات الخارجية.
6) إضافة تسجيل خطأ واضح عند فشل تحميل Supabase أو Mammoth.
7) تغيير مفاتيح cache-busting إلى 1.0.8 لضمان وصول التحديث للمستخدمين.

ملاحظة صريحة:
لم يتم تقسيم الحزم الكبيرة تلقائياً لأن النسخة مبنية كسلسلة أكواد تعتمد على ترتيب التنفيذ؛ تقسيمها دون build system واختبارات وظيفية كاملة قد يعطل المدير أو الطلبات. الإصلاح الحالي يحسن التحميل المتكرر والاستقرار بدون حذف وظائف.
