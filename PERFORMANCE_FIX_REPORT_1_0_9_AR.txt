منصة آلين v1.0.9 — تحسين الأداء الآمن

التغييرات:
1) تقديم ملفات CSS قبل JavaScript لبدء رسم الواجهة بسرعة أكبر.
2) تأجيل تنفيذ Supabase وحزم التطبيق إلى ما بعد بناء الصفحة، مع حفظ ترتيب التنفيذ.
3) إلغاء تحميل مكتبة Mammoth عند كل زيارة وتحميلها فقط عند فتح معاينة DOCX.
4) تخفيف التخزين الأولي في Service Worker ومنع تنزيل حزم الكمبيوتر والموبايل معاً.
5) تصحيح استجابة Service Worker حتى لا يعيد HTML عند فشل ملف JavaScript أو CSS.
6) توحيد إصدار الواجهة والإعدادات والحزم والتخزين المؤقت إلى 1.0.9.

التحقق:
- 54 ملف JavaScript: لا توجد أخطاء صياغة.
- جميع مراجع الملفات المحلية موجودة.
- تشغيل صفحة الكمبيوتر على Edge: ناجح.
- تشغيل صفحة الموبايل على Edge: ناجح.
- لا توجد أخطاء SyntaxError أو ReferenceError أو TypeError أثناء اختبار الفتح.

الأثر المتوقع:
- ظهور الواجهة أسرع في الزيارة الأولى.
- عدم تنزيل مكتبة DOCX إلا عند الحاجة.
- تقليل تنزيلات تثبيت Service Worker الأولية بحوالي 2.7MB غير مضغوطة.
- بقاء التخزين المؤقت للملفات التي يستخدمها الجهاز فعلياً.
